html5magzine

html5杂志 微周刊 杂志

左右滑动翻页 上下滑动浏览页面 长按显示左侧导航栏

使用分栏适配页面 iphone 分栏数 = 1 ipad 分栏数 = 3

页眉 页脚 左侧导航栏(默认隐藏)

关于杂志内容
图片必须指定高度
